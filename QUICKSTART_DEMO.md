# クイックスタートガイド - デモモード

## 🎭 Azureなしでエンドツーエンドデモを体験

### 1. バックエンドを起動

```bash
cd api/FunctionsApp
func start
```

起動時に以下が表示されればOK:
```
🎭 DEMO MODE: Using mock services (no Azure connection required)
```

> **Note**: `local.settings.json` の `DEMO_MODE` は既に `"true"` に設定済みです

### 2. フロントエンドを起動

```bash
cd frontend
pnpm install  # 初回のみ
pnpm dev
```

### 3. ブラウザでアクセス

1. `http://localhost:3000` を開く
2. 左サイドバー下部のトグルスイッチを確認
3. **🎭 デモモード** が選択されていることを確認

### 4. デモ機能を試す

#### セッション一覧を表示
- サイドバーの「セッション」をクリック
- 2件のサンプルセッションが表示されます

#### セッション詳細を表示
- セッションをクリック
- 文字起こし、要約、感情分析結果を確認できます

#### 音声アップロード（シミュレーション）
- サイドバーの「音声アップロード」をクリック
- ファイルをドラッグ&ドロップ
- デモモードでは実際のアップロードなしで成功レスポンスが返ります

## 本番モードへの切替

### バックエンド

1. `api/FunctionsApp/local.settings.json` を編集:
```json
{
  "Values": {
    "DEMO_MODE": "false",
    "CosmosDbConnectionString": "YOUR_CONNECTION_STRING",
    ...
  }
}
```

2. Functions を再起動

### フロントエンド

左サイドバー下部のトグルスイッチで **☁️ 本番モード** を選択

## トラブルシューティング

### Azureエラーが出る
→ `DEMO_MODE` が `"true"` （文字列）になっているか確認

### モックデータが表示されない
→ ブラウザのLocalStorageで `appMode: "demo"` を確認

### 本番モードに切り替わらない
→ Functions を再起動してください
